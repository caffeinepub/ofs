# Specification

## Summary
**Goal:** Simplify the profile interface by removing principal ID display and avatar customization options.

**Planned changes:**
- Remove principal ID display field from ProfilePage
- Remove avatar URL input field and related UI elements from ProfilePage
- Remove principal ID display from MobileMenu profile section
- Replace avatar images with display name initials or placeholder icons across profile-related components

**User-visible outcome:** Users will see a cleaner profile interface showing only their display name and settings, without principal ID or avatar customization options. Profile displays will use initials or simple placeholders instead of custom avatars.
